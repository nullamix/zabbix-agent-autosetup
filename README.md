# zabbix-agent-autosetup
This project provides scripts and configuration files to facilitate the offline installation and setup of Zabbix Agents in environments without internet access. It is designed to work in air-gapped infrastructures where security and isolation are paramount.

## Compatibility

This project has been tested and verified only up to Zabbix version 5.4.

The provided scripts and instructions are intended for installing the Zabbix Agent on various Ubuntu versions.

Functionality on newer Zabbix releases or non-Ubuntu distributions is not guaranteed.
